Чтобы запустить один браузер (чисто для проверки)
-------------
Выполняем команды
```sh
make set-os-<имя операционой системы (linux | macos | win)>
make set-env
python test_area.py
```

Чтобы запустить grid
-------------
Выбрать браузер:
CHROME:
```sh
make set-chrome

```
или FIREFOX:
```sh
make set-firefox

```
Запустить грид:
```sh
make grid

```
Запустить браузеры:
```sh
make node

```
Запустить тесты:
```sh
make run-tests

```
