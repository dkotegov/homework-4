### How to install
```bash
$ pip3 install requirements.txt
```

### How to run
```bash
$ ./grid.sh & ./node.sh & # Run selenium-server and add chrome and opera drivers
$ PASSWORD='********' BROWSER='CHROME' ./run_all_tests
```
