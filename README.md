## Чек-листы

### Размещение заказа. Управление заказом

#### Создание заказа

- Создать заказ с валидными данными
- Нельзя создать заказ с указанием невалидных данных
    - Ошибка при вводе слишком большого бюджета (больше 6-и знаков)
    - Нельзя прикрепить больше 5-и тэгов
    - Нельзя создать заказ без заполнения обязательных полей

#### Управление заказом на странице заказа

- Опубликовать заказ
- Снять с публикации (закрыть) заказ
- Ошибка при открытии страницы удаленного заказа

#### Редактрование заказа

- Изменить заказ с валидными данными
- Нельзя измененить заказ с указанием невалидных данных
    - Ошибка при вводе слишком большого бюджета (больше 6-и знаков)
    - Нельзя прикрепить больше 5-и тэгов
    - Нельзя изменить заказ без заполнения обязательных полей

#### Управлением заказом на странице списка заказов пользователя

- Опубликовать заказ
- Снять с публикации (закрыть) заказ
- Удалить заказ
